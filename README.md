# Twitter Bot
❗ Un bot Twitter totalement fait en Python par moi, et en français.

Fonctions:

  - Faire un Tweet
  - Supprimer un Tweet
  - Retweet
  - Unretweet
  - Follow
  - Unfollow
  - Bloquer
  - Débloquer
  - Masquer
  - Démasquer

__Dépendences :__

```pip install colorama```
```pip install tweepy```
